# Share.js

一个chrome插件，用于一键分享网页的，非常简单清新，无广告，无依赖


## 安装

 - Chrome市场：[Share.js](https://chrome.google.com/webstore/detail/sharejs/hknmadpbjapcmmggkpndokongeeibgep)
 - 下载本项目代码自行安装
